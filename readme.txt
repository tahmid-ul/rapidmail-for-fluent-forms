<a href="https://wordpress.org/plugins/fluentform">Fluent Forms</a> is a free and fastest form builder plugin for WordPress. This Plugin integrates with Rapidmail by providing an option for your users to signup in Rapidmail email newsletter lists while submitting a form in WordPress.

=== FEATURES ===
- Allow your users to sign up for a Rapidmail newsletter list using a Fluent Forms
- You can capture first name, last name, and (of course) email
- You can signup users to as many lists as you like
- You can set up the confirmation email via Rapidmail

== Installation ==
This section describes how to install the plugin (Rapidmail For Fluent Forms) and get it working.
Install From WordPress Admin Panel:

1. Login to your WordPress Admin Area
2. Go to Plugins -> Add New
3. Type "Rapidmail For Fluent Forms" into the Search and hit Enter.
4. Find this plugin Click "install now"
5. Activate The Plugin
6. Use Rapidmail on your site form the FluentForms module dashboard.


= Manual Installation =
1. Download the "Rapidmail For Fluent Forms" plugin from WordPress.org repository
2. On your WordPress admin dashboard, go to Plugins -> Add New -> Upload Plugin
3. Upload the downloaded plugin file and click Install Now
4. Be sure your FluentForms plugin is activated already.
5. Activate "Rapidmail For Fluent Forms" from your Plugins page.
6. Use Rapidmail For Fluent Forms on your site form the FluentForms module dashboard.
